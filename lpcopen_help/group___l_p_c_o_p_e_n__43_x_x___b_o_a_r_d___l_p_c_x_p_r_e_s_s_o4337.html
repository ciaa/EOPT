Jumper configuration
The list below shows the default jumper configuration used for all NGX Xplorer 4330 examples. Use this configuration for the examples unless an example alters this default setting in it's description.

  • Jumper JP1: OPEN
  • Jumper JP2: 1,2 SHUNT
  • Jumper JP3: OPEN
  • Jumper JP4: CLOSED
  • Jumper JP5: CLOSED

Optional UART setup
The Xpresso board (for now) do not include a default UART port for output of debug messages during code operation. The debug messages by default will be sent to UART0, the pins of which can be used from J2 Pin-0 [RX], J2 Pin-1 [TX] connecting FTDI cable TTL-232R-3V3 will get the UART output to a console.

Board recovery using the ISP pin
If the board's FLASH can no longer be programmed, the image in FLASH may be corrupt. To recover the board, the board's boot source can be altered by pressing and holding ISP push button SW2. .
