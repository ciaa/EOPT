Jumper configuration
The list below shows the default jumper configuration used for all Keil 4357 evaluation examples. Use this configuration for the examples unless an example alters this default setting in it's description.

  • J2 : ALL OPEN
  • J3 : 3V3
  • J5 : CLOSED
  • J9 : ALL OPEN
  • J10 : OPEN
  • J14 : DAC
  • J13 : UART3
  • J16 : UART3
  • J17 : OPEN
  • J18 : OPEN
  • J19 : CLOSED
  • J20 : CLOSED
  • J21 : CLOSED
  • P2_9 : L
  • P2_8 : L
  • P1_2 : L
  • P1_1 : H
  • 3V3A : CLOSED
  • 3V3R : CLOSED
  • 3V3IO : CLOSED

Board recovery using the ISP pin
If the board's FLASH can no longer be programmed, the image in FLASH may be corrupt. To recover the board, hold down the ISP pin when resetting the board or during the program sequence. The button can be released once the board programming cycle has started.
