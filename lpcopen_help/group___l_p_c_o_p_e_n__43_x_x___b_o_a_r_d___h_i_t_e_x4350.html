Board recovery using the ISP pin
If the board's FLASH can no longer be programmed, the image in FLASH may be corrupt. To recover the board, the board's boot source can be altered by changing the boot source on jumpers BOOT1 via BOOT4. Optionally, you can boot using ISP by resetting the board with the ISP button held down.
