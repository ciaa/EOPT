LPC4370 NXP LPCLink2 board support software API functions

Jumper configuration
For the LPCOpen examples, jumper JP1 should be installed and jumper JP2 should be removed.

Board recovery using the ISP pin
If the board's FLASH can no longer be programmed, the image in FLASH may be corrupt. To recover the board, remove jumper JP1, reprogram the board, and then replace jumper JP1.
